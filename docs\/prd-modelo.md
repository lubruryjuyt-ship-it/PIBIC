1. Identificação do Projeto
Título do Projeto:
Plataforma Interativa de Educação em Saúde: Efeitos dos Alimentos e Elementos de Liga no Corpo Humano
Área temática:
Educação, saúde e tecnologia.
Descrição breve (até 5 linhas):
O projeto consiste no desenvolvimento de um website educativo interativo voltado para estudantes do Instituto Federal de Goiás (IFG). A plataforma permite que os usuários interajam com um modelo ilustrado do corpo humano, arrastando alimentos e elementos de liga para compreender seus efeitos na saúde. O objetivo é promover aprendizagem sobre digestão, metabolismo e elementos químicos de forma visual e interativa, incentivando escolhas alimentares mais conscientes. 

2. Contextualização e Justificativa
A alimentação possui papel fundamental na saúde humana, porém muitos adolescentes realizam escolhas alimentares baseadas apenas em preferências pessoais, sem considerar os impactos dessas decisões no organismo. Além disso, a exposição a elementos químicos presentes em produtos do cotidiano também pode trazer consequências para a saúde. Nesse contexto, torna-se importante desenvolver estratégias educativas que auxiliem os estudantes a compreender melhor os efeitos dos alimentos e substâncias químicas no corpo humano.
Diante disso, o desenvolvimento de um website interativo surge como uma ferramenta pedagógica para promover educação em saúde de forma acessível e dinâmica. A proposta junta conhecimentos de biologia, tecnologia de materiais e programação, permitindo que os alunos aprendam por meio da interação com conteúdos digitais. Dessa forma, o projeto contribui tanto para a formação acadêmica dos estudantes quanto para a conscientização sobre hábitos alimentares mais saudáveis e escolhas mais responsáveis relacionadas à saúde

3. Objetivo Geral
Desenvolver um website interativo com a finalidade de informar e educar sobre os efeitos dos alimentos e dos elementos de liga no corpo humano, voltado para estudantes do Instituto Federal de Goiás, utilizando tecnologias de desenvolvimento front-end (HTML5, CSS3 e JavaScript), visando promover a conscientização sobre hábitos alimentares saudáveis e estimular o aprendizado interativo sobre saúde e ciência.

4. Objetivos Específicos
Desenvolver a estrutura do website utilizando HTML5 de forma organizada e semântica.


Criar um layout responsivo e visualmente atrativo utilizando CSS3.


Implementar funcionalidades interativas utilizando JavaScript.


Apresentar informações educativas sobre digestão, metabolismo e elementos de liga no corpo humano.


Promover uma navegação intuitiva e interativa por meio da manipulação de elementos visuais no site.

5. Público-Alvo
Faixa etária: 14 a 18 anos.


Perfil do usuário: Alunos do Câmpus Valparaíso de Goiás do Instituto Federal de 
Goiás (IFG).


Nível de letramento digital: Intermediário.


Principais necessidades do usuário: O usuário necessita de uma plataforma que apresenta informações sobre alimentação de forma dinâmica e visualmente atraente, facilitando a compreensão de hábitos saudáveis.

7. Requisitos Funcionais
Os requisitos funcionais descrevem o que o sistema deve ser capaz de realizar.
O sistema deve permitir a navegação entre as páginas do site.


O sistema deve validar os campos obrigatórios dos formulários.


O sistema deve exibir mensagens de erro quando necessário.


O sistema deve apresentar layout adaptável a diferentes dispositivos (responsivo).



8. Requisitos Não Funcionais
Os requisitos não funcionais descrevem características de qualidade do sistema.
O site deve ser responsivo.


O tempo de carregamento das páginas deve ser inferior a 3 segundos.


O código deve estar organizado em pastas separadas (HTML, CSS e JavaScript).


O projeto deve seguir boas práticas de indentação e nomenclatura.


O layout deve priorizar a legibilidade e a acessibilidade.
